# Complete Web Design Specification for Multiverse Portal

## Overall Architecture & Theme
**Concept**: Futuristic multiverse portal with cyberpunk/sci-fi aesthetics, featuring layered backgrounds, particle systems, and immersive scrolling experience.

## Background System (Layered)

### Layer 1: Base Background
```css
background-color: #0a0a1a;
/* or */
background: radial-gradient(ellipse at center,
  rgba(10, 10, 26, 1) 0%,
  rgba(5, 5, 16, 1) 100%
);
```

### Layer 2: Matrix Background Component
**Purpose**: Falling code rain effect (Matrix-style)
```
- Characters: Random alphanumeric + symbols
- Color: #00ff88 (neon green) with opacity 0.3-0.8
- Font: Monospace, 12-16px
- Animation: Vertical scroll, varying speeds (3-8s)
- Density: 50-80 columns
- Canvas-based rendering
- z-index: 0-1
```

### Layer 3: Dynamic Background Component
**Purpose**: Animated particle system with geometric shapes
```javascript
Particles:
- Types: Circles, hexagons, triangles, lines
- Colors: Neon blue, cyan, purple (rgba with 0.1-0.4 opacity)
- Size: 1-5px (circles), 20-60px (shapes)
- Count: 80-150 particles
- Movement: 
  * Slow drift (x: ±2px/s, y: ±2px/s)
  * Parallax effect (3 depth layers)
  * Perlin noise paths or random walk
- Connections: Draw lines between nearby particles
  * Distance threshold: 100-150px
  * Line opacity: 0.1-0.3
  * Color: Inherit from particle color

Geometric Overlays:
- Wireframe grid (subtle, opacity: 0.05)
- Hexagonal patterns
- Circuit board traces
- Tech HUD elements

Canvas Implementation:
- Full viewport size
- RequestAnimationFrame loop
- Responsive resize handling
- z-index: 1-2
```

### Layer 4: Content Layer
```css
position: relative;
z-index: 10;
```

## Animations & Effects

### Keyframe Animations
```css
/* Gradient Shift */
@keyframes gradient-shift {
  0%, 100% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
}

/* Pulse Glow */
@keyframes pulse {
  0%, 100% { 
    opacity: 1;
    box-shadow: 0 0 20px currentColor;
  }
  50% { 
    opacity: 0.6;
    box-shadow: 0 0 40px currentColor;
  }
}

/* Float Animation */
@keyframes float {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-20px); }
}

/* Rotate Slow */
@keyframes rotate-slow {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}

/* Fade In Up */
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(30px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* Border Flow */
@keyframes border-flow {
  0% { border-image-source: linear-gradient(0deg, colors); }
  100% { border-image-source: linear-gradient(360deg, colors); }
}
```

### Scroll Animations
```javascript
Intersection Observer setup:
- Threshold: 0.1-0.3
- Trigger: Add 'visible' class when in viewport
- Animation: fadeInUp, stagger children by 100-200ms
- Easing: cubic-bezier(0.4, 0, 0.2, 1)
```

## Responsive Breakpoints
```css
/* Mobile: < 640px */
- Single column layouts
- Reduced font sizes (50-70% of desktop)
- Simplified animations
- Touch-friendly spacing (48px minimum)
- Particle count reduced by 50%

/* Tablet: 640px - 1024px */
- Two column grids
- Medium font sizes (75-85% of desktop)
- Standard animations
- Particle count reduced by 30%

/* Desktop: > 1024px */
- Full multi-column layouts
- Full font sizes
- All effects enabled
- Full particle density
```

## Performance Optimizations
```javascript
- Use CSS transforms over position changes
- will-change: transform on animated elements
- RequestAnimationFrame for smooth 60fps
- Lazy load images and components
- Debounce resize/scroll handlers
- Reduce particles on low-performance devices
- Use CSS containment: layout style paint
```

## Accessibility Considerations
```css
/* Respect reduced motion */
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}

/* High contrast adjustments */
@media (prefers-contrast: high) {
  - Increase border widths
  - Boost color saturation
  - Remove subtle effects
}

/* Focus states */
:focus-visible {
  outline: 3px solid #00d4ff;
  outline-offset: 4px;
  box-shadow: 0 0 0 3px rgba(0, 212, 255, 0.3);
}
```
